# Foresto - Research Project Website

A simple, professional research project website for **Foresto** (Project ID: 25_26J_393), focused on AI-enabled food demand forecasting, inventory optimization, supplier intelligence, and waste management for Sri Lankan restaurants.

## Live Deployment

- Website: https://foresto.netlify.app/

## Repository

- GitHub: https://github.com/anuradhajayathunga/ResearchX

## Project Overview

This website presents the complete research project in one place, including:

- Domain background and literature survey
- Research gap and problem statement
- Four integrated research components
- Timeline and milestones
- Documents and presentations
- Team and supervisor information

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Poppins)

1. Clone the repository.
2. Open the project folder.
3. Launch `foresto.html` in your browser.
